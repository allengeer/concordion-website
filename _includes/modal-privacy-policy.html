Like most website operators, Concordion.org collects non-personally-identifying information of the sort that web browsers and servers typically make available, such as the browser type, language preference, referring site, and the date, time and location of each visitor request. Concordion.org’s purpose in collecting non-personally identifying information is to better understand how visitors use its website. From time to time, Concordion.org may release non-personally-identifying information in the aggregate, e.g., by publishing a report on trends in the usage of its website.

This website uses Google Analytics, a web analytics service provided by Google, Inc. ("Google"). Google Analytics uses "cookies" to help the website analyze how users use the site. A cookie is a string of information that a website stores on a visitor’s computer, and that the visitor’s browser provides to the website each time the visitor returns. Concordion.org visitors who do not wish to have cookies placed on their computers should set their browsers to refuse cookies before using Concordion.org websites, with the drawback that certain features of Concordion.org’s websites may not function properly without the aid of cookies. To find out more about cookies, including how to see what cookies have been set and how to manage and delete them, visit www.allaboutcookies.org.

For more information on Google Analytics, see How Google uses data when you use our partners' sites or apps.

By using this website, you consent to the processing of data about you in the manner and for the purposes set out above.
